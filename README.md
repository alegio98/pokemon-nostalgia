Fix v3 Pokémon Nostalgia

Questo aggiornamento sistema:
- font Pokémon DS applicato anche a REGISTRAZIONE POKÉDEX, VOCE DEL POKÉDEX, CLASSIFICAZIONE e testo delle MOSSE
- menu in basso rosso Pokémon invece che arancione
- pulsanti e testi più grandi e leggibili
- nomi e numeri Pokémon ancora più grandi
- header lista cambiato in POKÉDEX / ROSSOFUOCO
- rimosso KANTO DATABASE
- cache aggiornata per forzare il refresh

Se sei già dentro la cartella del progetto:

unzip -o ~/Downloads/pokemon-nostalgia-fix-v3.zip -d .
python3 -m http.server 8000

Poi apri http://localhost:8000 e fai CMD + SHIFT + R due volte.
